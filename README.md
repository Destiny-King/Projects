# springboot-vue-demo

springboot+vue框架练手项目

## Project Structure

1.springboot：后端springboot框架代码

2.vue：前端vue框架代码

3.sql：MySQL数据库结构和数据。



## Project setup

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
